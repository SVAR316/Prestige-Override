# Prestige Conditions Override RU

Серверный мод для **SPT 4.0**.

Подтягивает требования на престиж ближе к **оригинальному Escape from Tarkov**: уровень, деньги, навыки, убежище и старт престиж-квестов — **без сюжетных квестов**, которых в SPT нет.

# Prestige Conditions Override ENG

Server mod for **SPT 4.0**.

Brings prestige requirements closer to **original Escape from Tarkov**: level, money, skills, hideout, and prestige quest start — **without story quests** that aren't available in SPT.
