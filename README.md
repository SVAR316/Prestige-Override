# Prestige Conditions Override

Серверный мод для **SPT 4.0**.

Подтягивает требования на престиж ближе к **оригинальному Escape from Tarkov**: уровень, деньги, навыки, убежище и старт престиж-квестов — **без сюжетных квестов**, которых в SPT нет.

## Нужно установить

- SPT **~4.0**
- [WTT CommonLib](https://forge.sp-tarkov.com/)
- [WTT Content Backport](https://forge.sp-tarkov.com/)
- [Archangel Content Backport Prestiges](https://forge.sp-tarkov.com/)

Без этих модов сервер не загрузит зависимости.
